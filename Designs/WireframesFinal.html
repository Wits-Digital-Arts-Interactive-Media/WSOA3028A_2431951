SECOND ITERATION

this was posted by Ivy Chepkwony on 16th March 2024

Wireframes

The wireframes have not changed a lot from the original planning. Only the homepage and subpages will change. The navigation pages remain the same essentially including the profile page.

HOMEPAGE:

The homepage is simple and now contains only contains 2 main components:

  • The title of the website will be on a "whiteboard", and users can drag the individual letter around within that space. Above the letters, a small guiding piece of text will tell the user to move the letters.
  • The list of hyperlinks to allow the user to explore other parts of the website.

    • Subpages:

    All subpages will have the same formatting. This time, when directed to a specfic blog for example, each section/heading of that blog will be a clickable image to view the text associated with that heading. The idea is for the user to interact with the images as if they are flipping cards over in a game. This will be implemented with JS and CSS.

    User Flow and Information Structure

    User Flow and Information Structure has not changed from first iteration.

    Styling

    Styling overall remains the same

    Colour Scheme

    Colours to be used include: green, white, black, purple, pink (all pastel). The green will be used to emphasize certain elements, the purple and pink will be used throoughout on buttons and as background colours. Text will remain black and general background will remain white.
